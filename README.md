# FastAPI - HTMX Template

### To install requirement

```
python3 -m venv venv
source venv/bin/activate

pip install -r ./requirements.txt
```

### To use tailwind

```
./tailwindcss -i static/css/input.css -o static/css/output.css --watch
```

### To start server

```
uvicorn main:app --reload
```

url: localhost:8000
